# Mistake Logs

Root-cause mistake entries produced during `dsa-tutor` and
`mock-interviewer` sessions, using the format and category taxonomy
defined in `.claude/skills/dsa-tutor/SKILL.md`.

An entry is added only once the learner has confirmed the actual root
cause — the tutor never invents one. Typos, fatigue slips, and one-off
syntax errors with no conceptual weight are not logged.

Organization: one file per pattern (e.g. `two-pointers.md`,
`binary-search.md`), new entries appended to the matching file.

Categories: `reasoning-error`, `pattern-recognition-error`,
`implementation-error`, `reading-error`, `math-error`, `off-by-one`,
`state-tracking-error`.

Before an OA or interview, re-read the antidote lines here — they are
the highest-yield revision material in this repo because they are
derived from your own confirmed failure modes.
