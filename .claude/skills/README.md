# Teaching Skills

Claude Code Agent Skills that turn this repo into a placement-prep tutor: Socratic coaching that protects productive struggle instead of handing over answers.

## Skills

| Skill | Purpose | Origin |
|-------|---------|--------|
| `dsa-tutor` | Socratic DSA coaching — hint ladder, code circuit breaker, mistake logging | Adapted from think-before-code + workspace integration |
| `mock-interviewer` | Resume/company-driven mock rounds (technical, HR, managerial) | Original, interview discipline adapted from think-before-code |
| `aptitude-coach` | Timed quant/reasoning/pseudocode drilling | Original |
| `oa-prep-builder` | Company Prep HQ dashboards + predicted OA mock papers | Original |
| `problem-decoder` | Precise problem reading before solving | think-before-code |
| `dry-run-coach` | Real state-table tracing mechanics | think-before-code |
| `complexity-coach` | Big-O derivation drills | think-before-code |
| `debug-coach` | Bug isolation without rewritten solutions | think-before-code |
| `test-case-coach` | Learner-designed, justified test suites | think-before-code |
| `pattern-transfer-coach` | Turning solved problems into transferable patterns | think-before-code |
| `code-review-coach` | Guided code-review judgment practice | think-before-code |
